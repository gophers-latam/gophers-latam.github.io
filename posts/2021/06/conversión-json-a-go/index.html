Conversión JSON a Go

tools golang

Uno de los formatos de datos que se encuentra con mucha frecuencia en programación, principalmente web, es JavaScript Object Notation (JSON).

Cuando se recibe datos en formato JSON de representación, debe convertirse a un formato que sea fácil de manipular en el programa en Go. Hay algunas herramientas en línea que permiten convertir fuente JSON directamente en estructuras Go, aunque haciendo una simple búsqueda google.com/search?q=json+to+go se ve varidad de resultados, las destacadas o pioneras son:

JSON-to-Go - mholt.github.io/json-to-go/

Transform - transform.tools/json-to-go

Json2Struct - json2struct.mervine.net/

Por último como extensión de navegador para mejor accesibilidad existe json to go para Chrome.
